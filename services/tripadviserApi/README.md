# Data Ingestion Activies

## Overview
This directory contains the scripts that running and pulling the data from Tripadvisor-API.

Which are includes:

- (1) Checking the availability of the API
- (2) Pulling the data in json format 
- (3) Transform the data
- (4) Write down to the MongoDB as documents inside the collections

Following with the ingestion, there are 4 collections of involve with these activities:
- (1) hotelInfo.collection.locationDetails
- (2) hotelInfo.collection.nearbySearch
- (3) hotelInfo.collection.locationPhotos
- (4) hotelInfo.collection.locationReviews
